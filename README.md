# Azure Identity & Access Management Security Lab

This project demonstrates an Identity and Access Management (IAM) environment built using Microsoft Azure and Microsoft Entra ID.

The lab simulates enterprise identity management and security controls.

## Technologies Used

Microsoft Azure  
Microsoft Entra ID  
PowerShell  

## Security Features Implemented

• User identity lifecycle management  
• Role-based access control (RBAC)  
• Multi-Factor Authentication (MFA)  
• Conditional Access policies  
• Identity monitoring and sign-in logs  
• PowerShell automation for user provisioning

## Lab Objectives

Build a secure identity environment and demonstrate cloud IAM security practices used in enterprise environments.
